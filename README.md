# SampleProject

This project is a full-stack web application inspired by delhiver.com, built with React (frontend), Node.js/Express (backend), and MongoDB (database).

## Structure
- `client/`: React frontend
- `server/`: Express backend

## Features
- User authentication
- Order management
- Shipment tracking
- Admin dashboard

## Getting Started
1. Install dependencies:
   - Frontend: `cd client && npm install`
   - Backend: `cd server && npm install`
2. Start the backend: `cd server && npm start`
3. Start the frontend: `cd client && npm start`

## Database
- MongoDB is used for data storage. Configure your MongoDB URI in `server/.env`.

## Development
- Frontend runs on [http://localhost:3000](http://localhost:3000)
- Backend runs on [http://localhost:3001](http://localhost:3001) (update as needed)
